# 法定相続分シミュレーター

ご家族構成を入力すると、民法に基づく**法定相続分**（誰がどれだけ受け取れるか）を自動計算するWebアプリです。

> ⚠ 本ツールの結果はあくまで**参考値**です。実際の相続手続きでは、弁護士・司法書士・税理士等の専門家にご確認ください。

## 特徴

- ✅ 質問に答えていくだけのウィザード形式（専門用語不要）
- ✅ クリックで開く**用語解説**＋**Q&A**＋**図解**
- ✅ 計算結果を**関係図**としても表示
- ✅ **印刷／PDF出力**に対応（ブラウザの印刷機能を利用）
- ✅ 入力内容は**お使いの端末にのみ保存**（サーバー送信なし）
- ✅ **URL共有**機能（家族構成データをURL内に格納、サーバー保存なし）

## 対応している計算ケース

- 第1順位：配偶者＋子（直系卑属の再代襲は無限）
- 第2順位：配偶者＋直系尊属（父母→祖父母と世代単位で繰り上がる）
- 第3順位：配偶者＋兄弟姉妹（甥姪は1代限り）
- 配偶者のみ／血族のみ
- 半血兄弟姉妹（民法900条4号但書）
- 普通養子・特別養子
- 相続放棄（系統消滅・代襲なし）
- 相続欠格・廃除（代襲発生）
- 数次相続（再帰的に展開）
- 同時死亡の推定（民法32条の2）

## 使い方

1. 「サンプルから始める」で典型ケースを読み込むか、Q1から順に入力
2. 質問に答えていくと、画面下に**結果**（取り分一覧／関係図／計算根拠／民法条文）が表示されます
3. 必要に応じて「💾 この端末に保存」「🔗 URLでシェア」「🖨 印刷／PDFとして保存」を利用

## プライバシーについて

- 入力されたご家族構成のデータは、**サーバーには一切送信されません**
- ブラウザのローカルストレージ（お使いの端末内）にのみ保存されます
- URL共有機能を使った場合、データはURL内に含まれますが、URL自体はクライアント側で生成され、サーバーには送信されません

## 開発

ビルド不要の単一HTML構成です（React + Tailwind CSS via CDN）。

```
索引：
├ index.html       # メインUI（React + JSX）
├ calc.js          # 計算ロジック（純粋関数、UMD）
├ glossary.js      # 用語辞書・Q&A
└ test.cjs         # 計算ロジックの自動テスト（Node.jsで実行）
```

### テスト実行

```bash
node test.cjs
```

43件のテストケースで計算ロジックを検証しています。
